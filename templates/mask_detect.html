Face Mask Detector

This web application is used to detect whether a person is wearing face mask or not, while making sure it is put on the right way

This is the inspiration of this project, MaskDetector is flask web application powered with deep learning and computer vision, It is a face mask detector that detects if someone is wearing a face mask correctly, incorrectly or not. The system extracts an image and classifies it into 3 sections, No Mask, Correct Mask and Incorrect Mask.
